# Web Application

A modern web application built with Angular, featuring a knowledge base and live logs monitoring capabilities.

## Features

- **Ticket Viewer**: Ticket viewer interface
- **Knowledge Base**: Documentation and information repository
- **Live Logs**: Real-time log monitoring interface
- **Responsive Design**: Built with Tailwind CSS for modern, responsive layouts
- **Markdown Support**: Powered by ngx-markdown for rich content

## Prerequisites

- Node.js (v18 or later recommended)
- npm (v9 or later) or yarn
- Angular CLI (v20 or later)

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/inono05/webpage.git
   cd webpage
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server***
   ```bash
   ng serve --host 0.0.0.0 --port 4200
   ```
   The application will be available at `http://localhost:4200/`
